# Installation
    ```sudo make clean install```
# Run
    ```exec dwm```
# Configuring
    Actual config is - "config.h".
# Create Desktop Entry
    path: ```/usr/share/xsessions/dwm.desktop```
```
    [Desktop Entry]
    Encoding=UTF-8
    Name=dwm
    Comment=Dynamic window manager
    Exec=dwm
    Icon=dwm
    Type=XSession
```
